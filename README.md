# Responsive Form with SASS and Flexbox

__Standards:__

- HTML5 
- CSS 3
    - FlexBox
    - Responsive Web Design:
        - Media Queries
- SCSS
- SASS
- Javascript

* **Minifying and compiling scss** - gulp sass

* **Minifying, compiling scss and watch function** - gulp
